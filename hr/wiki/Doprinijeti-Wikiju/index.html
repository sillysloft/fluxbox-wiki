Doprinijeti ovom Wikiju

This is a community wiki and it grows with contributions from people like you. This is not a classic type wiki where you can just login and edit articles. These pages are automatically generated, static websites, hosted by gh-pages.io. But contributing is still quite easy, it does require a certain amount of enthusiasm though, specifically you have to use a so called version control system named git. so go checkout gh-link and edit /wiki-content and create a pr. Thank you very much.
